TTL: 5 minutes\n- Cleared on: logout, manual invalidation\n\n### localStorage Cache (Persists across tabs):\n- User profile with metadata\n- Max age: 5 minutes\n- Cleared on: logout, cache expiration\n\n---\n\n## 📝 Developer Notes\n\n### When Does Caching Help Most?\n- User logs in → page refresh → cache restores instantly\n- User switches tabs → comes back → uses cache instead of re-fetching\n- Multiple rapid auth operations → shares cached data\n\n### When Is Cache Invalidated?\n- User logs out (automatic)\n- User updates profile/roles (should invalidate manually)\n- Cache expires (5 minutes)\n- Manual call: `authSessionCache.invalidate(userId)`\n\n### Connection Detection\n```javascript\n// Supported connection types:\nnavigator.connection.effectiveType\n// \"4g\" → 10s timeout\n// \"3g\" → 20s timeout  \n// \"2g\" / \"slow-2g\" → 30s timeout\n// saveData: true → 20s timeout (regardless of speed)\n```\n\n---\n\n## ⚠️ Important Notes\n\n1. **No Breaking Changes**:
